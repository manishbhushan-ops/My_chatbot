## ğŸš€ How to Get Started

Follow these steps to install and run the project locally:

### Prerequisites
Make sure you have:
- **Python 3.8+** installed
- Internet connectivity to access OpenAI API

### Installation Instructions

1. **Clone the Repository**  
   ```bash
   git clone https://github.com/manishbhushan-ops/My_chatbot.git
   cd My_chatbot
   ```

2. **Install Dependencies**  
   ```bash
   pip install -r requirements.txt
   ```

3. **Add OpenAI API Key**  
   - Open the file `.streamlit/secrets.toml` and add your API key:
     ```toml
     [api_keys]
     openai_api_key = "your_openai_api_key_here"
     ```

4. **Run the Application**  
   Launch the app:
   ```bash
   streamlit run app.py
   ```

5. **Open in Browser**  
   Navigate to `http://localhost:8501` or the URL provided in your terminal.
